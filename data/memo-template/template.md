# Initial Assessment Memo — Template

> This template structures the *Initial Assessment Memo* in line with IDW PS 210 / ISA 250.
> It is the authoritative structure the NOCLAR agent follows.
> **All fields are mandatory** unless marked otherwise.

---

## 1. Header

| Field | Value |
|---|---|
| **Client** | *<Name of the audited entity>* |
| **Engagement ID** | *<internal identifier>* |
| **Reporting period** | *<Fiscal year / period>* |
| **Responsible partner** | *<Name>* |
| **Memo author** | *<Name, function>* |
| **Date** | *<YYYY-MM-DD>* |
| **Confidentiality level** | Confidential / Strictly confidential |
| **Memo version** | 1.0 |

## 2. Trigger of the memo

What led to the creation of this memo? Source, timing, form
(whistleblower, the audit team's own observation, external information).

## 3. Facts of the case

Factual account in chronological order. Distinguish between
**documented, verifiable facts** and **currently unconfirmed
statements**. Every factual claim with a source reference
(e.g. invoice XY, witness statement Z dated DD.MM.YYYY).

## 4. Persons involved and their functions

|---|---|---|---|
| | | | |

## 5. Legal assessment — potentially violated provisions

Multiple entries possible. For each offence:

- **Norm / provision:** *<e.g. § 299 StGB, § 30 OWiG, § 334 StGB i. V. m. IntBestG>*
- **Elements of the offence:** *short assessment of whether the elements prima facie apply to the facts*
- **Risk classification per IDW PS 210:** indirectly / directly relevant

## 6. Materiality for the financial statements

Expected effects on the financial statements (qualitative and quantitative):

- *Direct effects* (provisions, contingent liabilities, adjustment of income-statement items);
- *Indirect effects* (going-concern risk, reputational damage, concession / debarment risks).
